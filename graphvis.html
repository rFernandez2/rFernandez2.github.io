Graph Visualizer for Competitive Coding.

Chrome Browser Recommended.

This a simple in-browser app to display a graph. The data is entered in famous competitive coding format.

First line - contains two space-separated integers denoting number of vertices and edges.

In the following lines (amount of lines equal to the number of edges).

Each of the next lines will be containg 2 or 3 integer first being the source node, then destination node and 'optional' weight of the edge.

REFRESH the page to draw another graph.

Here are some examples.
